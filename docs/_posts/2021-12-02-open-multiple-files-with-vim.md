在命令行中，用`vim`可以一次性以不同方式打开多个文件。

用`-o`以**水平方式**打开文件:
```
$ vim -o file1 file2
```
显示效果为：
```
+++++++++++++++++++++++++++++
|                           |
|          file1            |
|                           |
|---------------------------|
|                           |
|          file2            |
|                           |
+++++++++++++++++++++++++++++
```

用`-O`以**竖直方式**打开文件：
```
$ vim -O file1 file2
```
显示效果为：
```
+++++++++++++++++++++++++++++
|            |              |
|            |              |
|            |              |
|   file1    |     file2    |
|            |              |
|            |              |
|            |              |
+++++++++++++++++++++++++++++
```

用`-p`以**标签方式**打开文件：
```
$ vim -p file1 file2
```
显示效果为：
```
+++++++++++++++++++++++++++++
|   tab1  |  tab2  |        |
|---------------------------|
|     |        |            |
|     v        v            |
|   file1    file2          |
|                           |
|                           |
|                           |
+++++++++++++++++++++++++++++
```

以上。
